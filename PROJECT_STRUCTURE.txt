        │

            ├── pipedrive.js      # Pipedrive CRM API
            ├── monday.js         # Monday.com GraphQL API
            ├── airtable.js       # Airtable REST API
            └── emailFallback.js  # FormSubmit or console logging

│   ├── *.jpg                     # Optimized images
│   ├── index-[hash].css          # Minified CSS (~22KB)

│   └── vendor-[hash].js          # React & libraries (~264KB)

Component Hierarchy
===================

App
├── Navbar (sticky header)
├── Hero (with parallax)
├── About (two-column layout)
├── Programs (4-card grid)
├── Testimonials (3 glass cards)
├── Gallery (responsive grid)
├── CTA (parallax background)
├── RegistrationForm (lead capture)
├── Footer (social links)
└── VideoModal (conditional render)

Data Flow
=========

User Fills Form
      ↓
RegistrationForm.jsx
      ↓
submitLead() → LeadService.js
      ↓
Selects Provider (from env)
      ↓
      ├─→ webhook.js → Generic Webhook API
      ├─→ hubspot.js → HubSpot Forms API
      ├─→ pipedrive.js → Pipedrive API
      ├─→ monday.js → Monday.com GraphQL
      ├─→ airtable.js → Airtable API
      └─→ emailFallback.js → FormSubmit or console
      ↓
Success/Error Response
      ↓
Display Message to User

Scripts
=======

npm run dev        → Start Vite dev server (port 3000)
npm run build      → Build for production (output: dist/)
npm run preview    → Preview production build locally
npm run lint       → Run ESLint on all files
