## הוראות שימוש

1. **התקנה**:
```bash
npm install
```

2. **הגדרת CRM** (`.env`):
```env
VITE_CRM_PROVIDER=webhook
VITE_CRM_WEBHOOK_URL=your-webhook-url
```

3. **הרצה**:
```bash
npm run dev
```

4. **בנייה**:
```bash
npm run build
```
